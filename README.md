# Web_Sec

웹 해킹 공격 기법과 보안 취약점에 대해 공부한 내용을 정리하는 저장소입니다.

## 목적

이 저장소는 웹 보안 학습 과정에서 다룬 공격 기법, 취약점 원리, 실습 환경, 대응 관점의 내용을 체계적으로 기록하기 위해 만들었습니다.

## 정리할 주제

- SQL Injection
- XSS
- CSRF
- File Upload Vulnerability
- Command Injection
- Path Traversal
- Authentication / Authorization Bypass
- Session / Cookie Security
- Web Shell
- SSRF
